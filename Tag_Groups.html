Tag Taxonomy

Tag groups used in QCR rankings.

Tags add context that a single grade cannot capture, including stability, quality patterns, access, pricing, reputation, activity, size, status, MTL usage, and proofreading.

Stability Tags

Inconsistent
Quality swings with no clear pattern.
Consistently-Inconsistent
Quality swings with a clear, repeatable pattern.
Consistent
No large swings in quality.
Recently Improved
Quality improved recently.
Recently Declined
Quality declined recently.
Series-Dependent
Quality changes depending on series.
Genre-Dependent
Quality changes depending on genre.

Case-by-Case Quality Tags

  • Unreadable
  • Terrible
  • Poor
  • Barely Readable
  • Okay
  • Decent
  • Good
  • Great
  • Excellent
  • Official-Quality

Joke Tags

Schrodinger's Translation
Both readable and not readable at the same time.
"Readable"
Technically readable if you have unusually high reading comprehension.
Tripping Hazard in Hell
A lighthearted warning tag for especially rough translations.

Judgment Tags

Avoid at all costs
Avoid even if it is the only option available.
Fallback Only
Better than alternatives, not actually good.
Read at own Risk
Can be decent, but very inconsistent.
Good in a Pinch
Acceptable when better options are not available.

Access Tags

Paywalled
All translations are paywalled.
Latest Chapters Paywalled
Only latest translations are paywalled.
Subscription-Based
Translations are only free through a subscription.
Split Chapters
Chapters are split into multiple parts by the translator or translation group.

Pricing Tags

Cheap
$0.50 or more per chapter.
Average
$1 or more per chapter.
Expensive
$2 or more per chapter.
Very Expensive
$3 or more per chapter.
Ridiculously Expensive
$4 or more per chapter.

Reputation Tags

Highly-Regarded
Has a near spotless reputation for quality and consistency.
Well-Regarded
Has a good reputation for quality and consistency.
Regarded
Has neither a good nor bad reputation for quality and consistency.
Controversial
Has a mixed or controversial reputation for quality and consistency.
Unknown
Too new to have an established reputation or does not otherwise have one.

Translation Standard Tags

Gold Standard
Always has perfect or almost perfect translations.
Above-&-Beyond
Always has excellent translations.
Above-Standard
Always has good or great translations.
Standard
Always has okay or decent translations.
Sub-Standard
Always has barely readable or worse translations.

MTL Usage Tags

MTL-Light
Uses some machine translation.
MTL
Often uses a lot of machine translation.
MTL-Heavy
Always uses a lot of machine translation.

Proofreading Tags

Maximum-Proofreading
Always uses extensive proofreading.
More-Proofreading
Uses enough proofreading to make it readable.
Standard-Proofreading
Uses enough proofreading to make it understandable.
Minimal-Proofreading
Uses very little proofreading.
No-Proofreading
Almost never uses proofreading.
Bad-Proofreading
Uses poor proofreading.
Terrible-Proofreading
Uses very poor proofreading.

Activity Tags

Defunct
Group shut down or disbanded.
Semi-Active
Only works on one series at a time and never works on multiple series at once.
Low-Activity
Currently has only one active series.
Active
Group active on 2-10 series.
High-Activity
Group active on 11 or more series.

Size Tags

Individual
Only 1 person.
Small Group
2-4 people.
Average Group
5-9 people.
Large Group
10 or more people.
Company
Translation company. Used as a practical placeholder until more non-group structure tags exist.

Status Tags

Unofficial-Unpaid Translator
Translated by unpaid fans.
Unofficial-Paid Translator
Translates on commission. Gets paid to translate but is not official.
Official Translator
Official translation group or company.
Unofficial-MTL Translator
Exclusively uses MTL to mass-produce low-quality translations with no real proofreading.
